# federated-api

This repo serves as an example of how to use cargo workspaces with docker to run microservices.

## Tools and technology
- Postgres
- Rust
  - async_graphl _(graphql server)_
  - poem _(http server)_
  - diesel _(ORM)_
- GraphQL (Apollo Federation)
- Docker
